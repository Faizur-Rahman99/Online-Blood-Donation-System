# Online Blood Donation System

## Welcome to LifeBlood: Your Pathway to Saving Lives through Blood Donation

The **Online Blood Donation System** is a web-based platform designed to connect **blood donors, blood seekers, hospitals, and blood banks** in a unified and efficient ecosystem. The platform simplifies blood donation and blood request management by enabling quick donor matching, campaign organization, and real-time blood stock tracking.

The primary goal of this project is to improve accessibility to blood during emergencies and reduce delays in locating compatible donors.

---

## Features

### User Registration & Approval
- Registration for:
  - Donors
  - Seekers
  - Hospitals
  - Blood Banks
- Admin approval required for account activation
- Verification-based secure onboarding

---

### Admin Dashboard
- Manage all user registrations
- Approve or reject accounts
- Monitor platform activity
- Verify user information

---

### Donor Dashboard
- View blood donation requests
- Accept or decline requests
- Update personal profile
- Track donation history
- Availability-based participation

---

### Blood Bank Dashboard
- Create and manage donation campaigns
- Maintain blood stock by blood group
- Receive requests from hospitals and seekers
- Forward requests to suitable donors

---

### Hospital & Seeker Dashboard
- Send blood requests
- Track request status
- Monitor request history

---

### Search & Matching System
- Search donors by:
  - Blood group
  - Location
  - Availability
- Automatic blood type compatibility matching
- Only eligible donors (last donation >120 days) appear in search

---

### Blood Donation Campaign Management
- Create blood collection campaigns
- Define:
  - Location
  - Time
  - Target blood groups
- Donors can register for campaigns

---

### Notification System
- New donation request alerts
- Campaign updates
- Approval notifications

---

### Privacy & Security
- Protected user information
- Secure access control
- Confidential medical data handling

---

## Problem Statement

Blood shortages during emergencies often occur due to delays in finding suitable donors and poor communication between hospitals, blood banks, and donors.

This platform addresses these challenges by creating a centralized digital system that makes blood donation faster, smarter, and more accessible.
